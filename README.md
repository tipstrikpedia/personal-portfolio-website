# Personal Portfolio Website

A simple personal portfolio website built with **HTML, CSS, and JavaScript**.

## Live Demo

After publishing with GitHub Pages, add your live website link here.

Example:

```text
https://tipstrikpedia.github.io/personal-portfolio-website/
```

## About This Project

This project is a beginner-friendly personal portfolio website. It introduces the owner, shows skills, displays GitHub projects, and provides contact information.

It is useful for building an online profile, practicing GitHub Pages, and supporting job or remote work applications.

## Features

- Responsive personal homepage
- About section
- Skills section
- Projects section
- Contact section
- Dark mode and light mode
- Smooth scrolling navigation
- GitHub project links

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## How to Use

1. Create a new repository named `personal-portfolio-website`.
2. Upload all project files.
3. Go to **Settings**.
4. Open **Pages**.
5. Select branch `main` and folder `/root`.
6. Save and wait for the GitHub Pages link.

## Customization

Before publishing, update these parts:

- Replace `your-email@example.com` with your real email.
- Edit the About section.
- Add more projects when available.
- Update the name if needed.
